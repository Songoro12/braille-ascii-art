# Braille ASCII Art Generator

This is a basic braille ASCII art generator program made with Python via OpenCV.


## Run

`braille.py <image> [-w width] [-t threshold] [-s] [-o output.txt]`

- **width:** Output width in characters. (Set -1 to original size)
- **threshold:** Threshold value to convert binary image.
- **swap:** Swap braille characters.
- **output:** Write the result to specified file.


**Note:** Some terminals may not support braille characters, you can redirect the output to txt file (with -o or --output) and view with text editor (suggested monospace font) or you can use **gui.py**

## Algorithm

Simply reads the image, converts it binary image and samples the pixels like 4x2 matrix as if they are braille ascii characters.
